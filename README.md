# YAJU&U Meme Coin Website

GitHub Pages 対応の静的サイトです。`index.html` をルートに置いているので、そのまま GitHub にアップロードできます。

## 使い方

1. このフォルダの中身を GitHub リポジトリにアップロード
2. GitHub の **Settings > Pages** から Branch を `main`、Folder を `/root` に設定
3. `config.js` の `CONTRACT_ADDRESS` と `DISCORD_URL` をローンチ後に差し替え
4. BGM を使う場合は `assets/bgm.mp3` を追加

## 注意

YouTube楽曲や人物素材は権利確認が必要です。公式サイトとして公開する場合は、許可済み素材・自作素材・著作権フリー音源の使用を推奨します。
