# YAJU&U NFT MOMENTS

NBA Top Shot風のミームNFTガチャ＆マーケットMVPです。GitHub Pagesでそのまま公開できます。

## 入っている機能
- 10連ガチャ
- ブラックアウト / カウントダウン / レア度別演出
- 1枚ずつ排出
- My Collection保存（localStorage）
- Phantom接続デモ
- NFT Mint導線デモ
- マーケット一覧、検索、レア度フィルター
- 出品/購入デモ
- Discord / Magic Eden / CA差し替え

## 編集する場所
`config.js` の CA、Discord、Magic Eden、Tensor、Collection Address を差し替えてください。

## 本番化に必要なもの
- Solana NFT Collection 作成
- Metaplex Candy Machine 接続
- Helius APIなどでNFT保有確認
- Magic Eden / Tensorへのコレクション連携
- Discord Bot / Collab.Landでホルダーロール付与

## 注意
現状はフロントMVPです。実際の売買・ミントにはスマートコントラクト/マーケットプレイス連携が必要です。
